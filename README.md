# Projeto - ESTAGIÁRIO(A) DELIVERY E BACKTEST

Este projeto foi desenvolvido a pedido Quod para a vaga de **ESTAGIÁRIO(A) DELIVERY E BACKTEST**

Neste repositório, você irá encontrar:
- Um Script em **Jupyter Notebook** (em Python)
- Um Script em **SQL**
- Um Documento em formato **.pdf**

## Script em Jupyter Notebook (.ipynb)
Ao acessar o Script em Jupyter Notebook, você irá encontrar o código referente à criação, manipulação e extração de dados, junto a algumas análises gráficas.  

Para rodar esse código, você pode baixá-lo e abri-lo pela plataforma **Google Colab**:  
1. Crie uma conta Google (ou entre se já possuir uma).  
2. Clique em **"file" → "upload notebook"**.  

Se preferir, pode rodar em Python. Clicando em **"file" → "download" → "import .py"** você exportará o arquivo como Python e poderá usá-lo da maneira que preferir.

## Script em SQL
Ao acessar o Script em SQL, encontrará queries de criação, inserção e consulta de dados referentes à tabela.

## Documento (.pdf)
Ao acessar o Documento, será encontrada uma análise sobre os insights e dados gerados nos Scripts do Notebook.
